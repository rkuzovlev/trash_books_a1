# TrashBooks shop

### Quick start

```bash
# clone our repo
$ git clone https://github.com/rkuzovlev/trash_books_a1.git trash_books

# change directory to your app
$ cd trash_books

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```

go to [http://localhost:8080](http://localhost:8080) in your browser.
